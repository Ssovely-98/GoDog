# Outline
강원도 지역의 반려동물 관광과 관련된 정보를 제공하여 반려동물 동반 관광 여행을 쉽게 계획할 수 있도록 돕는 정보들을 제공한다. 제공하는 정보는 관광지, 숙박, 식음료, 체험, 동물병원, 산책로 등으로 세분화하여 제공함
